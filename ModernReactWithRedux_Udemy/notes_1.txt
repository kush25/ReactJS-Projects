In react => App.js

App is a component.
React scan each and every <> tag and validates is it a DOM, if yes dispalys on browser. And if not but<> some component then goes to that component and checks what it is returning and then displays.

React is known as reconciler
ReactDom is known as renderer

useState is a function for working with Reacts 'state' system.
It keeps track on data that changes over time.
Help in updating HTML on the screen.


Creating react app:
npx create-react-app my-app

If you get any errors about missing templates or how a global Create React App install is no longer supported even when using this command, you likely need to remove the global package from your system:

npm uninstall -g create-react-app
